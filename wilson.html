W3C

  • W3C - World Wide Web Consortium

    El W3C es la principal organización internacional encargada de desarrollar y mantener estándares abiertos para garantizar el crecimiento a largo plazo de la Web. Fundada en 1994 por Tim Berners-Lee, creador de la World Wide Web, su misión es desarrollar protocolos y directrices que aseguren el crecimiento continuo de la web.

    Algunos logros clave del W3C:

    • HTML - Lenguaje de marcado para la creación de páginas web.
    • CSS - Hojas de estilo en cascada que definen la presentación de las páginas web.
    • XML - Lenguaje de marcado extensible que facilita la interoperabilidad entre sistemas.
    • WCAG - Pautas de accesibilidad web que promueven la inclusión digital para personas con discapacidad.

    El W3C también gestiona otras tecnologías críticas como WebAssembly, WebRTC y el estándar para gráficos SVG. Su objetivo es que la web sea accesible, internacional y utilizable por cualquier persona desde cualquier dispositivo.
